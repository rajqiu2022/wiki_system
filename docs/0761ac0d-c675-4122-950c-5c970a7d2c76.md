# Maduino Zero SIM808 GPS Tracker

## 1. Introduction

Maduino Zero SIM808 GPS Tracker is an IoT (Internet of things) Solution based on the 32-bit AtmelГўВЂВ™s SAMD21 MCU and GPRS/GSM GPS module SIM808, which is the upgrade version of SIM900. It can be a real project for IoT projects such as smart-home, outdoor monitoring, shared bicycle, etc.
The Maduino Zero SIM808 GPS Tracker based on the Arduino, users can program it with Arduino IDE, which is very easy especially suit for the none-programmers. There are also guide for users to learn how to create the first IoT project with this board, with which the starters can learn the hardware and programming skill quickly.
This guide helps you with the basic usage of the Maduino Zero SIM808 GPS Tracker.

### 4.2 Hardware Connection

- Plug a MicroSIM card.
- Plug a GPS Antenna to designator which shows GPS.
- Plug a GSM Antenna to designator which shows GSM.
- Plug a Bluetooth Antenna to designator which shows BT.
- Plug a Micro USB Cable to Maduino Zero SIM808.
- Plug a 3.7V lithium battery to J6 or AC-DC Adapter to Maduino Zero SIM808
  \*\* > - Note1: A 3.7V lithium battery normal voltage is 3.7V, but the range voltage often from 3.4V to 4.2V
  > - Note2: Micro USB just used for communicate with the board, it doesnГўВЂВ™t supply power to the board because the SIM808 Module need a 2A pulse current when power on.
  > - Note3: The current of AC-DC Adapter should be 2A or more. Otherwise the SIM808 may not work or work unstable.
  > - Note4: Both the battery or the DC jack can be used for power supply. While DC connected, the board will be powered by DC, while DC removed, it will switch to battery automatically.\*\*

#### 4.3.1 How to get GPS Location information?

SIM808 Module is a GPRS/GSM+ GPS Quad-band module, with this board, we can use it as a GPS tracker, data acquisition, remote control and a lot of IOT project or smart home project. This example show you how to get GPS location.
Attention: To get GPS location, please ensure you are outdoor, and plug the GPS Antenna. Indoor may get the incorrect location or no location information.

1.Turn on GPS, using the AT command ГўВЂВњAT+CGNSPWR=1ГўВЂВќ to turn the GPS:

```
AT+CGNSPWR=1
```

2.Define the last NMEA sentence that parsed by AT command ГўВЂВњAT+CGNSSEQ=RMCГўВЂВќ

```
AT+CGNSSEQ=RMC
```

3.GNSS navigation information parsed from NMEA sentences, by AT commandГўВЂВќ AT+CGNSINFГўВЂВќ, then the Maduino GPS will begin to navigate the GPS info:

```
AT+CGNSINF
```

4.Turn on navigation data URC report, and report every GNSS FIX, by ГўВЂВњAT+CGNSURC=1ГўВЂВќ

```
AT+CGNSURC=1
```

![Maduino_Zero_SIM808 15 .jpg](https://www.makerfabs.com/image/wiki_image/2022-09-12_20_05_50_0.jpg)

If not get the GPS signal, it returns:

```
+CGNSINF: 1,0,19800105235944.000,,,,0.00,0.0,0,,,,,,0,0,,,,,
```

#### 4.3.3 Bluetooth test

- Before the Bluetooth test, we firstly can by means of the command of "AT+GMR" to inquire the firmware version of the SIM808 module, in order to promise it can use the Bluetooth function.
- You can get the code from here: [Github](https://github.com/Makerfabs/SIM808_Demo/tree/master/sim808_BLE)
- Connect an LED, the negative pole of LED is grounded, and the positive pole is connected to D8.
- Upload the code SIM808\_Demo/sim808\_BLE/sim808\_BLE.ino.
- Open the Serial Monitor, enter the following command to get the Bluetooth name and physical address.

```
AT+BTHOST?    Inquiry host device name
```

- Just use other Bluetooth devices to pair and connect, send "LED ON" to the module, and the LED will be on, and send "LED OFF", and the LED will be off. You can use the following two Bluetooth solutions to connect to the module.
