# MaPIE RP2040 0.96" OLED

## Introduction

We chose RP2040 as the main controller, which is a SoC with excellent performance and powerful functions and can be used in different development platforms.

At the same time, choose a 0.96" OLED to design some small projects for the developers to display on the screen. The battery module allows it to escape from the wires. I believe that this development board will let developers have a good experience.

Model: [MAPIEOLED](https://www.makerfabs.com/mapie-rp2040-0-96-inch-oled.html?search=2040)

![MaPIERP20400.96OLED.jpg](https://www.makerfabs.com/image/wiki_image/2023-04-03_01_03_52_0.jpg)

## Feature

- Controller: RP2040
- On board Flash: 16MB
- Power: USB Type-C; battery; On board charger
- Available pins: have 26 GPIO, 4 ADC
- Serial Wire Debug
- 0.96" OLEDï¼SSD1306/SSD1315ï¼, 30 PIN

### Usage with Arduino

**To ensure the running environment, the same version is recommended.**

1.Install the Arduino IDE V1.8.19.

2.Open the board manager, and install the **Arduino Mbed OS RP2040 Boards**

![MaPIERP20400.96OLEDconfig.jpg](https://www.makerfabs.com/image/wiki_image/2022-11-28_17_41_34_0.jpg)

![MaPIERP20400.96OLEDboard.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_17_41_47_0.JPG)

3.Select and set the parameter in the Tools menu, as the picture:

![MaPIERP20400.96OLEDconfig2.jpg](https://www.makerfabs.com/image/wiki_image/2022-11-28_17_44_03_0.jpg)

**Note: Different PC has different Serial COM number**

- How to display the demo under the Arduino development environment.

Get the code in the [GitHub](https://github.com/Makerfabs/MaPIE/tree/main/MaPIE-0.96-OLED), then program is called as**MaPIE\_RP2040\_0.96\_OLED\_V1.0**.

When **MaPIE RP2040 0.96" OLED** powered on for the first time, it pop up a window(a new disk called RPI-RP2), which is for the RP2040 to choose the **uf2 file** and decide development environment.

Download the firmware to initialize the board, and copy the **[MaPIE\_RP2040\_0.96\_OLED\_V1.0.ino.uf2](https://github.com/Makerfabs/MaPIE/tree/main/MaPIE-0.96-OLED/firmware)** to the board. It can let us use Arduino for developing.

![MaPIERP20400.96OLEDRPIRP2.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_18_10_25_0.JPG)

Verifing and uploading the [MaPIE\_RP2040\_0.96\_OLED\_V1.0.ino](https://github.com/Makerfabs/MaPIE/tree/main/MaPIE-0.96-OLED). When Finshing upload, you will see the Serial Monitor as follows.

![MaPIERP20400.96OLEDarduino.jpg](https://www.makerfabs.com/image/wiki_image/2022-11-28_18_32_48_0.jpg)

### Usage with Micropython

In this part, display the demo under the Micropython development environment. You can get the code in the [GitHub,](https://github.com/Makerfabs/MaPIE/tree/main/MaPIE-0.96-OLED), then we will use the **Micropython** file.

Due to using the Micropython, We should install **Thonny** for developing.

- Download Thonny at: <https://thonny.org/>, select the operating system you are using.(Window/Mac/Linux)

![MaPIERP20400.96OLEDinatall thorny.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_19_39_54_0.JPG)

- Open the Thonny IDE(you may need to select the language the first time you open the IDE),
  select the Tools->Options:

![MaPIERP20400.96OLEDinatall thorny config.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_19_41_14_0.JPG)

- In the interpreter sheet, select MicroPython(Raspberry Pi Pico), and select the option Try to
  detect port automatically, and select OK.

![MaPIERP20400.96OLEDinatall thorny option.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_19_49_42_0.JPG)

- Click the button stop, and the Shell will show the MicroPython version and shows that the
  MicroPython is installed successfully.

![MaPIERP20400.96OLEDthornyready.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_19_50_43_0.JPG)

- Download the latest MicroPython firmware uf2 file at: <https://micropython.org/download/rp2-pico/>, **Press&Hold the button BOOTSEL on the board, and power up MaPIE RP2040 0.96" OLED with the type-C cable to PC. Then open the PC, there will be a new disk called RPI-RP2, copy this uf2 file into the new disk.**

![MaPIERP20400.96OLEDthornyversion.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_22_31_39_0.JPG)

- When the board restart. Right-click the file and upload the code to the board.

![MaPIERP20400.96OLEDthornyupload.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-28_22_34_06_0.JPG)

![MaPIERP20400.96OLEDthornyupload finsh.JPG](https://www.makerfabs.com/image/wiki_image/2022-11-29_00_01_09_0.JPG)

- Click the Run, the code will begin to run there, and the display on the development board will workï¼

![MaPIERP20400.96OLEDthornyrunning.jpg](https://www.makerfabs.com/image/wiki_image/2022-11-28_22_41_10_0.jpg)
