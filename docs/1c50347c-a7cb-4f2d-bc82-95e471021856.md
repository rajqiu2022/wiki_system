# ESP32-S3\_Parallel\_TFT\_3.16\_ST7701S

## 1.Introduction

This cute 3.16" display is for IoT applications, it features WiFi& Bluetooth 5.0 + BT mesh, with ESP32-S3R8 Dual-core LX7 microprocessor, integrates high-capacity Flash and PSRAM. The Onboard 3.16 inch LCD display(320Ã820 Pixels) can smoothly run GUI programs such as LVGL.

This product also integrates Onboard RTC chip, 6-axis IMU, MIC, Speaker and TF card slot, Lithium battery header, etc. Reserved USB, UART, and I2C interfaces for flexible expansion. It is suitable for the quick development of the HMI and other ESP32-S3 applications.

## 2.Features

- Controller: ESP32-S3
- Wireless: WiFi& Bluetooth 5.0
- LCD: 3.16", 820x320 resolution
- LCD Driver: ST7701S
- LCD interface: RGB+SPI
- Flash: 16MB Flash
- PSRAM: 8MB
- Touch Panel: No
- USB: 1 \* USB\_native
- Power Supply: USB Type-C 5.0V(4.0V~5.25V)
- Button: Flash button and reset button
- Expansion interface: JST1.0mm\*4
- Camera: No
- MicroSD: Yes
- RTC: Yes
- Charger: Yes
- LCD Backlight control: Yes
- Battery Connector: Yes
- Microphone input: INMP441
- Audio output: MAX98357(3.2W@4Î©)
- Motion Sensor: 3-axis gyroscope and 3-axis accelerometer QMI8658C
- Arduino support: Yes
- LVGL support: Yes

## 4.Usage in Arduino IDE

Install the Arduino IDE V1.8.19/V2.3.6 If you havenât installed the ESP32 Board SDK yet, follow the steps in this [guide](https://wiki.makerfabs.com/Installing_ESP32_Add_on_in_Arduino_IDE.html) to get started quickly.

For the ESP32-S3 Development board version, we recommend using versions that have been verified, such as 2.0.16, which is more stable, and less prone to errors.

Note: Different computers may have different port numbers when connecting to a development board. Please select the correct port number based on the development board you are connecting to.

Please download the relevant driver libraries before using these demos.

Download the Arduino GFX library from [github](https://github.com/Makerfabs/ESP32-S3_Parallel_TFT_3.16_ST7701S/tree/main/examples/arduino/lib/GFX_Library_for_Arduino).

Once the download is complete, copy these libraries into the Arduino library.

- The default is usually in the "C:\Users\Your username\Documents\Arduino\libraries"

**Note:** All demo work requires the involvement of the ESP32S3 Controller, and all of the following routines are programmed by burning code into the ESP32S3 Controller.

### 4.3 Display image

This project will automatically detect images on an SD card and display them on the screen, switching automatically every 2 seconds.

Note:

- The image dimensions must be 320 x 820 pixels.
- Currently only supports JPG format.

Use a card reader to save the images you want to display onto an SD card or you can save the default images on [GitHub](https://github.com/Makerfabs/ESP32-S3_Parallel_TFT_3.16_ST7701S/tree/main/examples/arduino/SD_Card/image).

- Install JPEGDecoder v2.0.0 library

![JPEGDecoder v2.0.0.png](https://www.makerfabs.com/image/wiki_image/2025-11-07_15_34_51_0.png)
