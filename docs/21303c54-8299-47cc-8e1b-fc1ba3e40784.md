## 1. Gateway Network Configuration

Connect the antenna and power adaptor to the gateway.
The power LED will show in red, and in about 15s, the indicator on the top will flash green, indicating that the gateway is booting.

![1.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_36_36_0.png)

There are two ways to connect to the Internet. Choose the one that works for you.

### 1.1 Connect with Ethernet

Connect the Ethernet cable to the Ethernet port, and the indicator on the top will show solid green if the gateway is successfully connected to the internet.

### 1.2 Connect with WIFI via Luci

- Step 1: Turn on the device AP hotspot

Press the button for 5s until the blue indicator flashes slowly to enter the configuration mode.

- Step 2: Connect to the AP hotspot

AP hotspot name is SenseCAP\_XXXXXX (6-figure MAC address),default password is 12345678; connect your computer to this AP hotspot.

- Step 3: Get your device Username and Password

You can find the Username and Password on your device label.

- Step 4: Log in to the Local Console

Input the IP Address (192.168.168.1) in your browser to enter the Local Console. Then input your device username and password, and click the Login button.

- Step 5: Click on Network - Wireless.

![4.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_38_45_0.png)

Click on the Scan button to scan the WIFI.

![5.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_39_05_0.png)

Select your WIFI to join the network.

![6.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_39_22_0.png)

Submit the WIFI password, and then click Submit and Save.

![7.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_39_36_0.png)

Then click **Save and Apply** to apply your settings. The indicator on the top will show solid green if the gateway is successfully connected to the WIFI.

After powering on the gateway, check the gateway working status.

**LED Indicator:**

![8.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_40_38_0.png)

### 1.3 Connect Gateway to The Things Network(TTN)

TTN (The Things Network) forms a global, decentralized IoT network dedicated to providing extensive wireless communication coverage for IoT devices through community-supported LoRaWAN gateways. Next, we will guide you on how to successfully connect your gateway to the TTN network.

#### 1.3.1 TTN Configuration

- Step 1: Log into The Things Stack.

If you don't have a TTN account, please register first.

- Step 2: Register the gateway.

![10.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_43_53_0.png)

Enter your Gateway EUI and click "confirm".

![11.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_44_36_0.png)

Enter your Gateway ID, Gateway ID, Frequency plan and click "Register gateway".

![12.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_44_47_0.png)

You can check the Gateway in the overview after successful registration.

#### 1.3.2 Gateway Configuration

Configure the gateway via the Web UI, please to log into Luci page first.

- Step 1: LoRa Network Settings

Navigate to LoRa --> LoRa Network.

![13.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_46_25_0.png)

- Step 2: Set Mode to Packet Forward.

Click "Save&Apply" to apply your settings.

![14.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_47_13_0.png)

- Step 3: Channel Plan Settings

Navigate to LoRa --> Channel Plan.

![15.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_47_37_0.png)

Select the Region and Frequency plan according to the actual choice.

After setting, click Save& Apply.

![16.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_01_48_03_0.png)

By Now, the Gateway setting been completed.

## 3. Data Visualization with Thingspeak

The Things Network is the largest LoRaWAN data server, but it do not support data storage& visualization, to achieve this, we utilize [ThingSpeak](https://thingspeak.mathworks.com/) for data presentation and analysis.

- Step1: Create a Channel:

Log in to ThingSpeak.

Click on 芒聙聹My Channels芒聙聺.

Click on 芒聙聹+ New Channel芒聙聺.

Fill in the Channel name and field names (e.g., Field1: Temperature, Field2: Humidity).

Click on 芒聙聹Save Channel芒聙聺.

![26.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_00_15_0.png)

- Step2: Obtain the Write API Key茂录職

Open the newly created Channel.

Click on the 芒聙聹API Keys芒聙聺 tab.

Copy the 芒聙聹Write API Key芒聙聺.

![27.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_00_55_0.png)

- Step3: Configure Payload Formatter茂录職

In the device details page, click on the 芒聙聹Payload Formatters芒聙聺 tab.

Click on the 芒聙聹Uplink芒聙聺 tab.

Select 芒聙聹Custom JavaScript Formatters芒聙聺.

Here JavaScript code to parse the binary data sent from the device. You can download & Copy all AgroSense JavaScript at [github](https://github.com/Makerfabs/Agrosense-Decoder).

![5.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_03_20_0.png)

Finally click "Save changes".

![28.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_04_01_0.png)

- Step4: Configure Webhook at TTN

In the application page, click on the 芒聙聹Integrations芒聙聺 tab.

Select 芒聙聹Webhooks芒聙聺.

Click on 芒聙聹+ Add webhook.

![29.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_04_34_0.png)

Fill in the following information and click on 芒聙聹Create ThingSpeak webhook芒聙聺.

| Webhook ID | Webhook ID |
| --- | --- |
| Channel ID | Your ThingSpeak Channel ID |
| API Key | Your ThingSpeak Write API Key |

![30.png](https://www.makerfabs.com/image/wiki_image/2024-11-28_02_06_23_0.png)

With the above setting, TTN forward the AgroSense data to thingspeak for visualization analysis.

Now Press AgroSense LoRaWAN脗庐 Sensor's User Button, you will successfully see the data in Thingspeak.

**NOTE:** Agrosense sensor send data every1 hours by default, but you can press the user button in your implement of the project, to help you quickly check if the whole project works as intended.

Agrosense support Data Downlink to change the time interval of reporting, check [Downlink](https://wiki.makerfabs.com/AgroSense_LoRaWAN%C2%AE_Sensor_Instruction_Manual.html#6-downlink) for the details.
