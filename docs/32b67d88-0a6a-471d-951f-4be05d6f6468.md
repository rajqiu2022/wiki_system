# Maduino Zero NFC

## 1. Introduction

Maduino Zero NFC is an IoT (Internet of things) Solution based on the 32-bit Atmel's SAMD21 MCU and NXP Near Field Communication Controller PN532, which is can be used for pay applications, entrance guard system, etc.

The Maduino Zero NFC based on the Arduino, users can program it with Arduino IDE, which is very easy especially suit for the none-programmers. There are also guide for users to learn how to use this board, with which the starters can learn the hardware and programming skill quickly.

This guide helps you with the basic usage of Maduino Zero NFC.

Model: [OAC532NFC](https://www.makerfabs.com/maduino-zero-nfc.html)

### 3.1 Install Software for the Arduino/Genuino Zero

The following screens show how to install software for the Maduino(Arduino/Genuino) Zero boards in the Arduino IDE.

1.Open the Boards Manager From the top Arduino IDE menu, select Tools-> Board-> Boards Managerâ¦ to open the Boards Manager dialog box.

![Arduino_Board_V1.0.png](https://www.makerfabs.com/image/wiki_image/2022-12-10_01_33_47_0.png)

2.Select Arduino SAMD Boards and Install In the **Boards Manager** dialog box, type **Arduino Zero** into the search field to easily find the Arduino SAMD package for 32-bits ARM Cortex-M0+ boards as shown in the image below. Install it.

![Arduino_BoardManager_V1.0.png](https://www.makerfabs.com/image/wiki_image/2022-12-10_01_53_42_0.png)

3.Find the Arduino Zero Boards on the Arduino IDE Menu After the software package has been installed, the new Arduino boards can be seen on the Arduino IDE Board menu found under **Tools âBoard** as shown in the image below.

![Arduino_Serial.png](https://www.makerfabs.com/image/wiki_image/2022-12-10_01_54_20_0.png)

A new section called **Arduino SAMD (32-bits ARM Cortex-M0+) Boards** can be seen on the Board menu which contains Arduino M0, M0 Pro, Zero, MKR boards and others. This means the right package has been installed and Maduino ready to go.

### 3.2 Hardware Connection

- Plug a NFC Antenna to designator which shows ANT1.
- Plug a Micro USB Cable to Maduino Zero NFC

![Maduino_Zero_NFC_03.jpg](https://www.makerfabs.com/image/wiki_image/2022-12-10_02_03_30_0.jpg)

- Note 1: You can supply power by a 3.7V lithium battery after you upload the program to the board.
- Note 2: a 3.7V lithium battery normal voltage is 3.7V, but the range voltage often from 3.4V to 4.2V
- Note 3: Both the battery or the MicroUSB can be used for power supply. While MicroUSB connected, the board will be powered by MicroUSB, while MicroUSB removed, it will switch to battery automaticly-.

### 3.3 Program the Maduino Zero

Select the desired **Arduino/Genuino Zero(Native USB port)** board from the Arduino IDE Board menu and then select the correct Port number from the menu item below Board on the Tools Menu. The Maduino board can now be programmed in the usual manner.

![Arduino_Zero_NativeUSB_V1.0.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_21_56_22_0.png)

Select Arduino/Genuino Zero(Native USB port) Open our programming code [iso14443a\_uid.ino](https://github.com/Makerfabs/MaduinoZeroNFC/tree/master/iso14443a_uid)

Click Verify button to compile it.

![VerifyButton01.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_22_18_26_0.png)

If there is no any errors, it will show Done compiling.

![Arduino_Done_Compiling_V1.0.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_22_18_57_0.png)

Click Upload button to upload firmware to the Maduino Zero NFC board.

![UploadButton01.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_22_19_41_0.png)

After done uploading, it will show the programming OK, as below:

![Arduino_Done_Uploading_V1.0.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_22_21_02_0.png)

From the top Arduino IDE menu, select **Tools â Serial Monitor**â¦ to open the Serial Monitor dialog box.

![SerialMonitor01.png](https://www.makerfabs.com/image/wiki_image/2022-12-11_22_21_33_0.png)

Select baud rate 115200, it will shows the board begin to works:
