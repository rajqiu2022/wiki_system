## 1. Introduction

This Sunton ESP32-S3 4.3ГўВЂВќ IPS provides a cheap solution for ESP32 S3 display, in intends for easy/ quick applications. It is based on the latest Dual Core CPU ESP32-S3(8M PSRM,16M Flash), the display is IPS RGB with resolution 800 \* 480, and Capacitive Touch.

## 2. Feature

- ESP32-S3-WROOM Controller
- Working voltage 4.75-5.25V
- Resolution: 800\*480 IPS
- Dimension: 4.3 inch
- Capacitive Touch
- Onboard SD card socket
- Extended IO for applications' usage
