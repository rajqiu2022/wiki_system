# SenseLora 4G gateway

## 1. Introduction

If you want the data to be transferred via 4G, either to a remote server or a simple SMS, select the SenseLoRa\_4G Gateway please. This Gateway is based on ESP32-S3, it has on-board LoRa receiver and a 4G module, so it get the LoRa data and then transmit the data via 4G to cloud server such as TTN, or the monitors phone with SMS.

![senselora_4g_lte_cat1_gateway1 1.jpg](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_02_59_0.jpg)

![SenseLoRa_4G_LTE_CAT1_Gateway1.jpg](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_03_06_0.jpg)

Model:[SL4GGY](https://www.makerfabs.com/senselora-4g-lte-cat1-gateway.html)

## 2. Features

- IP68 Waterpoof
- ESP32S3 controller, 16M flash/8M PSRAM
- High performance LoRa/4G antenna
- Power: DC 5V/3.7V battery

## 4. Transmit Air Monitor data to Datacake

If you are a plant lover, have a well-tended backyard garden, or run a small, self-sufficient farm, an efficient and accurate air monitor is an indispensable companion. While pursuing a green lifestyle, you must also pay attention to energy saving and economic considerations. LoRa technology, with its excellent low power consumption and low cost, fits your needs perfectly. The Makerfabs Senserlora is based on lora technology for small-scale farming. Makerfabs Senserlora currently consists of two monitors (Air monitor and Soil monitor) and two receivers (Receiver and 4G gateway). Now, I will show you the combination of Air monitor and 4G gateway and upload data to Datacake via 4G.

![20240903_19_13_22_0.jpg](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_08_54_0.jpg)

[Datacake](https://datacake.co/) is a versatile low-code IoT platform. It's easy to add some charts to the dashboard to make your data more visual.

![20240903_19_22_35_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_10_57_0.png)

### 4.1 Hardware

This project consists of three parts, firstly the lora sender, which is our air monitor, it will monitor the temperature and humidity data sent to the 4G gateway; Secondly the lora receiver, which is our 4G gateway, it will receive the data uploaded to datacake; And finally the data visualisation in datacake. In datacake we can display the data in a variety of forms that can be analysed.

![20240903_23_25_53_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_11_45_0.png)

[SenseLoRa\_Industrial-grade Air Monitor](https://www.makerfabs.com/senselora-industrial-grade-air-monitor.html) supports the LoRa protocol, which achieves low power consumption, long distance and security. It will monitor the data every hour (of course, you can change the monitoring interval yourself via wifi) and all the sensor data periodically via LoRa P2P, this LoRa signal can be received by SenseLoRa\_4G LTE CAT1 Gateway, and thus to display/record/analyze on PC.

![20240903_19_29_59_0.jpg](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_12_34_0.jpg)

[The SenseLoRa\_4G LTE CAT1 Gateway](https://www.makerfabs.com/senselora-4g-lte-cat1-gateway.html) has an on-board LoRa receiver and a 4G module, so it receives the LoRa data and then transmits the data via 4G to cloud server such as Datacake, or to the monitor's phone via SMS.

![20240903_19_20_40_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_13_14_0.png)

## 5. Transmit Air Monitor data to Thingspeak

This demo will show you use this device to transmit the temperature and humidity to Thingspeak that the T&H CAN be checked everywhere.

The user needs a Lora transmitter to transmit the corresponding signal to receive the temperature and humidity information. Let's take the example of a [SenseLora\_Industrial-grade Air Monitor](https://www.makerfabs.com/senselora-industrial-grade-air-monitor.html) **as a transmitter**.

### 5.1 Hardware

1.[SenseLoRa\_Industrial-grade Air Monitor](https://www.makerfabs.com/senselora-industrial-grade-air-monitor.html)

2.[SenseLoRa\_4G LTE CAT1 Gateway](https://www.makerfabs.com/senselora-4g-lte-cat1-gateway.html)

### 5.2 Thingspeak

- sign in or sign up for ThingSpeak

If you already have an account, log in and enter the password, otherwise click Create one to register.

![20240617_00_18_06_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_30_53_0.png)

- Create new channel

Select "Channel --> My Channels".

![20240617_00_29_22_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_31_30_0.png)

click "New Channel".

![20240617_00_30_28_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_31_38_0.png)

Write the name and field information and click Save Channel to create successfully.

![20240702_19_30_00_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_32_01_0.png)

### 5.3 Arduino

- Select "Tools > board:"xxx" > ESP32 Arduino > ESP32S3 Dev Module".

5.Verify the code and upload.

![20240617_02_34_40_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_36_43_0.png)

### 5.4 Result

Once the download is complete, power on the [SenseLora\_Industrial-grade Air Monitor](https://www.makerfabs.com/senselora-industrial-grade-air-monitor.html), press the reset button every time and wait about 1 minute to receive a message in Thingspeak.

![20240702_20_26_27_0.png](https://www.makerfabs.com/image/wiki_image/2024-09-15_03_37_45_0.png)

Such as adding information about carbon dioxide and light intensity.

```
float eco2 = doc["eco2"];
float lux = doc["lux"];
sprintf(url, "AT+HTTPPARA=\"URL\",\"http://api.thingspeak.com/update?api_key=%s&field1=%.1f&field2=%.1f&field3=%.1f&field4=%.1f\"",THINGSPEAK_APIKEY, temp, humi, eco2, lux);
```

In the **void lora\_receive\_task()** funtion, change the char url[200] to char url[600];

```
char url[600];
```

Add field in Thingspeak:

![Thingspeak add field1.png](https://www.makerfabs.com/image/wiki_image/2024-10-10_19_29_25_0.png)
