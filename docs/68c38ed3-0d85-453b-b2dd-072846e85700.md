# Maduino Zero SIM868 GPS Tracker

## 1. Introduction

We already have the [Maduino Zero SIM808 GPS Tracker](https://www.makerfabs.com/maduino-zero-sim808-gps-tracker-v3-5.html) which supports Bluetooth3.0+EDR, and now we would like to introduce a new product Maduino Zero SIM868 GPS Tracker which supports Bluetooth4.0 including BLE, and has more GNSS functions that supports GPS/BEIDOU/GLONASS Location.

## 4. Hardware connection

- Plug a MicroSIM card.
- Plug a GPS Antenna to designator which shows GPS.
- Plug a GSM Antenna to designator which shows GSM.
- Plug a Bluetooth Antenna to designator which shows BT.
- Plug a Micro USB Cable to Maduino Zero SIM808.
- Plug a 3.7V lithium battery to J6 or AC-DC Adapter to Maduino Zero SIM868

![IMG_20231030_141610.jpg](https://www.makerfabs.com/image/wiki_image/2023-10-29_23_18_48_0.jpg)

## 5. Usage

### 5.1 Get GPS location

- Open [demo](https://github.com/Makerfabs/Maduino-Zero-SIM868-GPS-Tracker/tree/main/example/Zero_SIM868_SD_HTTP_GPS_AT_Commands) by Arduino IDE.
- Verify the code and upload.
- Open the serial monitor, and you can see the information on GPS.

```
AT+CGNSPWR=1      //Turn on GPS
AT+CGNSSEQ=RMC    //Define the last NMEA sentence that parsed
AT+CGNSINF        //GNSS navigation information parsed from NMEA sentences
```

![5ACF3CE52856481f9D234C5D039584BB.png](https://www.makerfabs.com/image/wiki_image/2023-10-26_23_51_24_0.png)

### 5.2 Get SD card information

- Open [demo](https://github.com/Makerfabs/Maduino-Zero-SIM868-GPS-Tracker/tree/main/example/Zero_SIM868_SD_AT_Commands) by Arduino IDE.
- Plug the SD card into the SD card slot.
- Verify the code and upload.
- Open the serial monitor, and you can see the information on SD card.

### 5.4 Bluetooth test

- Open [demo](https://github.com/Makerfabs/Maduino-Zero-SIM868-GPS-Tracker/tree/main/example/Zero_SIM868_AT_Command_Test) by Arduino IDE.
- Verify the code and upload.
- Open the serial monitor,enter the following command.

```
AT+BTPOWER=1          // Turn on the Bluetooth power
AT+BTHOST?            // Query the Bluetoothâs name and address
AT+BTHOS=<Name>       // Change the name of Bluetooth
```

![image.png](https://www.makerfabs.com/image/wiki_image/2023-11-01_02_50_10_0.png)

- open the Bluetooth in the mobile phone, you can see a Bluetooth named SIM868BT.

![IMG_20231101_175250.jpg](https://www.makerfabs.com/image/wiki_image/2023-11-01_02_53_15_0.jpg)

- Click matching in the mobile phone.

![image.png](https://www.makerfabs.com/image/wiki_image/2023-11-01_03_00_49_0.png)
