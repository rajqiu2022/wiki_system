# Get start with ESP-IDF

## 1.Installation Visual Studio Code and ESP-IDF extension.

Go to the [Visual Studio Code](https://code.visualstudio.com/Download) and choose to download the installation package based on your PC.

![1.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_15_45_04_0.png)

Open the VSCode, click **Extension Manager**, search for **ESP-IDF** and click **Install**.

![7.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_15_56_22_0.png)

## 2.ESP-IDF Configuration

Click **ESP-IDF** to into the configuration, select **EXPRESS**.

![8.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_15_56_14_0.png)

Select the installation version, path, tool path, and finally click **Install** to install.

![9.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_15_56_53_0.png)

Wait for the installation to complete.

![10.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_15_57_17_0.png)

## 3.Build your first project

Click **Show examples**.

![11.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_16_02_16_0.png)

Select a simple routine.

![12.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_16_02_28_0.png)

Click **build** to test.

![13.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_16_02_53_0.png)

Successful compilation completes the installation.

![14.png](https://www.makerfabs.com/image/wiki_image/2025-09-28_16_03_16_0.png)
