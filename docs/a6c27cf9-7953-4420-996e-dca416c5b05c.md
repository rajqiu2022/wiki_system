# Maduino LoRa Radio

## 1. Introduction

Maduino Lora Radio is an IoT (Internet of things) Solution based on Atmel's Atmega328P MCU and Lora module. It can be a real project for IoT projects(especially long-range, low power application ) such as smart farmer, smart warehouse, smart community, etc.

The Maduino Lora Radio based on the Arduino, users can program it with Arduino IDE, which is very easy especially suit for the none-programmers. There are also a guide for users to learn how to create the first IoT project with this board, with which the starters can learn the hardware and programming skill quickly.
This guide helps you with the basic usage of Maduino Lora Radio.

Model:[OAC915MLR](https://www.makerfabs.com/maduino-lora-radio-433m-868m-915.html)

## 3. Usage

**Warning: Donât operate anything when the board supplied on (That is, plug or unplug the Antenna or Sensor, in case of short-circuiting that may burn the IC down.)**

We will use one of Maduino Lora Radio board transmit temperature and humidity via Lora radio, and another Maduino Lora Radio shows the things on OLED Screen after received the message.

Prepares:

| Item | Quantity |
| --- | --- |
| Maduino Lora Radio | 2 |
| Lora Radio Antenna | 2 |
| Micro USB cable | 2 |
| Dupont Line | 7 |
| DHT11 Sensor | 2 |
| 128\*32 OLED Display Screen | 1 |

**1.Open Arduino IDE, select Arduino Pro or Pro Mini in Boards**

![500pxMaduino_Lora_Radio_IDE.jpg](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_07_06_0.jpg)

**2.Select Atmega328P(3.3V,8MHz) in Processor and correct Port**

![Maduino_Lora_Radio_V1.5_03.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_08_21_0.png)

**3.Download the [RadioHead file](https://github.com/Makerfabs/MaduinoLoraRadio), click Add. ZIP.Library, select RadioHead.**

**4. Lara Radio Transmitter Programming:**

Connect the Maduino Lora board to DHT11 Sensor as:

| Maduino Lora Radio | Maduino Lora Radio |
| --- | --- |
| 3V3 | VCC(Grey Wire) |
| D4 | DATA(White Wire) |
| GND | GND(Black Wire) |

Plug Lora Radio Antenna and Micro USB Cable

![Maduino_Lora_Radio_cable.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_46_37_0.png)

Install DHT sensor library in Library Manager. If you had installed it, go to the next step.

![Maduino_Lora_Radio_dht_lib.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_47_26_0.png)

Open Program [TransmitterDHT11.ino](https://github.com/Makerfabs/MaduinoLoraRadio)

- If your board is Maduino Lora Radio 868Mhz, Modify the macro

```
 #define RF95_FREQ 868.0 from #define RF95_FREQ 433.0
```

![Maduino_Lora_Radio_feq_set.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_55_29_0.png)

Click the Verify button to compile it.

![Maduino_Lora_Radio_verify.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_18_57_29_0.png)

If there are no errors, it will show done compiling.

![Maduino_Lora_Radio_compiling.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_19_43_41_0.png)

Click Upload button to upload firmware to the Maduino Lora Radio board.

![Maduino_Lora_Radio_v1.5_10.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_19_45_19_0.png)

If it runs okay, open Serial Monitor we can see something like this:

![700pxMaduino_Lora_Radio_v1.5_11.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_19_45_51_0.png)

**5.Lora Radio Receiver Programming**

- Connector an OLED to Maduino Lora board as:

| Maduino Lora Radio | OLED |
| --- | --- |
| A4 | SDA |
| A5 | SCL |
| 3V3 | VCC |
| GND | GND |

Plug Lora Radio Antenna and Micro USB Cable.

![700pxMaduino_Lora_Radio_v1.5_12.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_21_38_32_0.png)

Install SSD1306 library in Library Manager. If you had installed it, go to next step.

![Maduino_Lora_Radio_v1.5_13.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_21_39_09_0.png)

Open the program [ReceiverOLED.ino](https://github.com/Makerfabs/MaduinoLoraRadio)

- If your board is Maduino Lora Radio 868Mhz, Modify the micro

```
 #define RF95_FREQ 868.0 from #define RF95_FREQ 433.0
```

![Maduino_Lora_Radio_v1.5_14.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_21_40_38_0.png)

Click Verify button to compile it.

![Maduino_Lora_Radio_v1.5_15.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_21_41_10_0.png)

If there no errors, it will show done compiling.

![Maduino_Lora_Radio_v1.5_16.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_21_53_15_0.png)

Click Upload button to upload firmware to the Maduino Lora Radio board.

![Maduino_Lora_Radio_v1.5_17.png](https://www.makerfabs.com/image/wiki_image/2022-12-06_22_02_31_0.png)

**6. Result**

The Transmitter will detect the humidity& Temperature and send the testing result via Lora 433/868Mhz frequency. And the receiver will receive the result and show them in the OLED. The max wireless transmission distance could be, as Makerfabs measure, up to 2km.
