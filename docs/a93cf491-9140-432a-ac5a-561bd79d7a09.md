# MaTouch\_1.9\_inch\_TFT\_Breakout

## 1. Introduction

The Makerfabs MaTouch 1.9" board has garnered significant interest among users. However, based on feedback received, there was a consensus that the breakout of the ESP32S3 pins would enhance the usability of the board for various applications.

In response to user demands, the Makerfabs MaTouch 1.9" board brings creative updates. The enhancement lies in the breakout of all GPIO pins. Now, users can prototype their projects using a breadboard and integrate the board as a "controller" with other compatible devices. This optimization simplifies the development process and expand the board's utility.

## 2. Features

- Controller: ESP32-S3
- Wireless: WiFi& Bluetooth 5.0
- LCD: 1.9", 320x170 resolution, driver ST7789V2
- LCD interface: 8 bit 8080
- Touch Panel Driver: CST816D
- USB: USB Native, Type-C
- Power Supply: USB Type-C 5.0V(4.0V~5.25V); 3.7V Lipo battery
- Button: Flash button and reset button
- MicroSD: Yes
- Arduino support: Yes
- MicroPython support: Yes
- Operation temperature: -40â to +85â

## 3. Usage in Arduino IDE

### 3.1 Software setup

For the best user experience, it's important to use the recommended version throughout your development process. All projects are based on the ESP32-S3 development board, guaranteeing higher compatibility and stability. If you haven't installed the ESP32 Board SDK yet, follow the steps in this [guide](https://wiki.makerfabs.com/Installing_ESP32_Add_on_in_Arduino_IDE.html) to get started quickly .

**For the ESP32-S3 Development board version, we recommend using versions that have been verified, such as 2.0.16ï¼ which is more stable, and less prone to errors.**

1.Install the [Touchlib](https://github.com/Makerfabs/MaTouch-1.9inch-TFT-Breakout/blob/main/lib/TouchLib.zip) library.

- Open the sketch>include library> ADD.ZIP library,choose the download path of TouchLib and unzip the library to the Arduino library file.

![MaTouch_ESP32S3ParallelTFT1.9.jpg](https://www.makerfabs.com/image/wiki_image/2023-05-06_02_36_45_0.jpg)

![GFX v1.3.1.png](https://www.makerfabs.com/image/wiki_image/2024-10-11_19_38_45_0.png)

![MaTouch_ESP32S3ParallelTFT1.9Inchconfig.jpg](https://www.makerfabs.com/image/wiki_image/2023-05-06_02_24_28_0.jpg)

### 3.2 Squareline\_demo

Wire the DHT11 sensor, button and led to the MaTouch\_1.9\_inch\_TFT\_Breakout ,and the product can interact with them by downloading the example.

Before verifing and uploading the sketch, please install the same libraries in the Arduino IDE.

1.Install the [Mf\_Lvgl](https://github.com/Makerfabs/MaTouch-1.9inch-TFT-Breakout/blob/main/lib/mf_Lvgl.zip) library.

- Open the sketch>include library> ADD.ZIP library
- Choose the download path of MF\_Lvgl and unzip the library to the Arduino library file.
- open the mf\_lvgl>>src>>lv\_conf.h, Modify the compile option to change 0 to 1ï¼ and save the configuration.

![MaTouch_ESP32S3ParallelTFT1.9sorceboard.jpg](https://www.makerfabs.com/image/wiki_image/2023-05-10_22_52_39_0.jpg)

2.Install the DHT11\_sensor\_ library.

- Open the sketch>include library> Manage Libraries,choose the install DHT11\_sensor\_library.

![image.png](https://www.makerfabs.com/image/wiki_image/2023-09-20_01_38_49_0.png)

3.Install the [UI](https://github.com/Makerfabs/MaTouch-1.9inch-TFT-Breakout/tree/main/example/squareline_demo/library/ui) file.The UI file is designed to can be downloaded to the example in the form of arduino libraries.

- Open the sketch>include library> ADD.ZIP library
- Choose the download path of UI and install it to the Arduino library file.

4.Open the [squareline\_demo](https://github.com/Makerfabs/MaTouch-1.9inch-TFT-Breakout/tree/main/example/squareline_demo) by Arduino IDE.

5.You can wire the hardware according to the pin definitions you are using, as show in image:

![image.png](https://www.makerfabs.com/image/wiki_image/2023-09-20_02_28_16_0.png)

- Connect the data pin of the DHT11 sensor to IO1.
- Connect the INC pin of the button to IO2.
- Connect the negative terminal of the LED to IO3.

6.Verify and upload the sketch.

7.After uploading, the screen will be displayed as follows.
