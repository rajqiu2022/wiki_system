## 2.ESP32/ESP32S3 Download Mode

Put the esp32/esp32s3 chip into download mode Steps:

1.Open the serial monitor

2.Hold down the FLASH/BOOT button

3.Hold down the RST button

4.Release the RST button

5.Finally release the FLASH/BOOT button

When the serial port shows **"waiting for download"**, it has successfully entered the download mode.

![3.png](https://www.makerfabs.com/image/wiki_image/2025-02-06_22_59_23_0.png)

Now close the serial monitor and re-uploading the code, it will work.
