# Maduino Zero WIFI ESP8266

## 1. Introduction

Maduino Zero ESP8266 is an IoT (Internet of things) Solution based on the 32-bit Atmelâs SAMD21 MCU and WiFi module ESP8266. It integrated a micro Controller ATSAMD21G18, WiFi module ESP8266. The Maduino Zero ESP8266 is based on Arduino, users can program it with Arduino IDE, which is very easy especially suited for the none-programmers. There are also guide for users to learn how to create the first IoT project with this board, with which the starters can learn the hardware and programming skill quickly. With this board, you will easy to add text, SMS, and data to your project. It is good for your smart home project or M2M and so on.

## 2. Features

- BAT Input Voltage: 3.4-4.2V
- ATSAMD21G18, 32-Bit ARM Cortex M0+
- Integrated Power Control System
- Support AT Command
- Support USB Power Charge
- Support Micro SD Card
- Interface: I2C/SPI/UART/18\*GPIO

## 3. Usage

### 3.1 Project\_1 WIFI

1.Connect

Plug in a USB cable or a 3.7v lithium battery, Turn on the switch.

![Maduino_Zero_WiFi_ESP8266_3 1.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_19_53_54_0.jpg)

2.Download

Download and open the [Serial\_WIFI\_v1.0.ino](https://github.com/Makerfabs/Maduino-Zero-WiFi-ESP8266-v1.6)

Select the board Arduino/Genuino Zero(Native USB Port) and the corresponding Port.

![Maduino_Zero_WiFi_ESP8266_4.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_36_00_0.jpg)

modify the SSID and PASSWORD to your WIF and password:

```
#define SSID "Makerfabs"  // your ssid
#define PASS "20160704"   //pass word
```

Then click to download

![Maduino_Zero_WiFi_ESP8266_11.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_39_45_0.jpg)

Open the serial port tool and view the printed IP address

Note: Maduino Zero WiFi ESP8266 v1.6 will print a piece of scrambled code after power, which is a normal phenomenon.

3.Show

Open the browser and enter the IP address after '+CIFSR:STAIP '

The page will display the 'Makerfabs '[esp8266\_at\_instruction\_set\_en.pdf](https://wikiadmin.makerfabs.com/prod-api/vue-admin-template/user/getFileDownload?file=esp8266_at_instruction_set_en_2023-03-05_22_05_22_0.pdf)

![Maduino_Zero_WiFi_ESP8266_7.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_52_24_0.jpg)

Finally, wait for the module to finish executing the AT+RESTORE command to RESTORE factory Settings:

![Maduino_Zero_WiFi_ESP8266_9.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_52_57_0.jpg)

### 3.2 Project\_2 Basic test of SD card

Insert SD card, plug in a USB cable or a 3.7v lithium battery. Turn on the switch.

![Maduino_Zero_WiFi_ESP8266_2.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_53_35_0.jpg)

Download and open the [Serial\_SD\_v1.0.ino](https://github.com/Makerfabs/Maduino-Zero-WiFi-ESP8266-v1.6)

The following information can be seen in the serial port to prove that the SD card function is OK.

![Maduino_Zero_WiFi_ESP8266_8.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_54_20_0.jpg)

If the SD card is not inserted, or the SD card is abnormal, the following information can be seen:

![Maduino_Zero_WiFi_ESP8266_12.jpg](https://www.makerfabs.com/image/wiki_image/2023-03-05_21_54_52_0.jpg)
