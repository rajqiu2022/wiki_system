# Guidance

This article is dedicated to providing users with a learning system of basic guidance, including downloading and installing the Squareline application, operating instructions for integrating with Matouch, using the Arduino IDE and solving common problems, to ensure that users can easily get started. In addition, this manual also summarises the troubleshooting and solutions to common problems, aiming to remove the obstacles that users may encounter in practice.
